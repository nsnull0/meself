Profile web - <https://yoseph.ws/>

## Install

```shell
sudo apt install ruby ruby-dev zlib1g-dev && \
gem install bundler && \
bundle install && \
bundle lock --add-platform x86-mingw32 x86-mswin32 x64-mingw32 java
```
